# CustomAnki AI SRS

Интеллектуальная система интервальных повторений для изучения языков, работающая на базе **Google Gemini API**.

## Основные возможности
- **AI-генерация:** Создание карточек по любой теме с помощью нейросети.
- **Умная проверка:** Gemini оценивает смысл твоего ответа, а не просто сравнивает текст.
- **Менеджмент колод:** Создание, удаление и выбор активной колоды.
- **Веб-интерфейс:** Современный UI, доступный с компьютера и смартфона.
- **Безопасность:** Вход через Google OAuth или временный ввод API ключа.

## Как запустить

### 1. Подготовка
Убедитесь, что у вас установлен Python 3.10+.

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка
Создайте файл `config.json` в корневой папке (на основе `config.json.example` или введите вручную):
```json
{
    "google_client_id": "ВАШ_ID",
    "google_client_secret": "ВАШ_СЕКРЕТ",
    "api_key": "ВАШ_GEMINI_KEY",
    "model_name": "gemini-flash-latest",
    "db_path": "custom_anki.db",
    "web_host": "0.0.0.0",
    "web_port": 5000
}
```

### 4. Запуск
```bash
python app.py
```
Откройте в браузере: `http://localhost:5000`

## Использование с телефона (Android)
1. Узнайте свой локальный IP-адрес (команда `ipconfig` в Windows).
2. Введите в браузере телефона: `http://ВАШ_IP:5000`.
3. Убедитесь, что ваш телефон и ПК в одной Wi-Fi сети.

## Стек технологий
- **Backend:** Python, Flask, SQLite.
- **AI:** Google Generative AI (Gemini 2.0/1.5).
- **Frontend:** HTML5, CSS (Bootstrap 5), JavaScript.
- **Auth:** Authlib (OAuth2).
